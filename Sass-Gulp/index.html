Script

En el package.json tenemos la seccion "scripts". El package.json puede ejecutar los binarios y almacenar las dependencias. agregamos sass al scrpit y agregamos q es lo q se va a ejecutar cuando mandemos a llamar a ese script. Eso manda a buscar si existe dicha carpeta en el node-modules. Hay q tener en cuenta q no todas las dependencias tienen binarios

"scripts": { "sass": "sass" },

El lado derecho es el binario, el izquierdo es el script q lo va a mandar a llamar

"scripts": { "sass": "sass src/scss" },

Lo q se agrega ahi es la ruta del archivo. no hace falta poner este ultimo. lo compila pero hay q indicarle donde guardar

"scripts": {"sass": "sass src/scss:build/css"},

Finalmente estos scrpits se llaman con npm run sass (npm run "nombre del scrpit")

Podemos agregarle --watch para q realice los cambios de forma automatica

"scripts": { "sass": "sass --watch src/scss:build/css" },

CRTL + C para terminar el watch
