Gulp

Permite tener multiples tareas y realizar diferentes acciones. Optimizamos imagenes, cambiar formatos, unir codigo de js, css. Necesitamos un workflow q automatice tareas.

Utilizamos npn i --save-dev gulp

Va a ser una dependencia de desarrollo al igual q la de sass

En la raiz del proyecto creamos el gulpfile.js q va a ser el contenedor de nuestras tareas

Se conecta package.json con gulpfile

Las tareas en gulpfile.js se llaman con npx gulp primerTarea

NPX se instala junto a node. Nos permite ejecutar paquetes sin necesidad de instalarlos globalmente

tambien podemos llamar a la funcion con npm. para eso debemos agregar lo siguiente al script del package.json:

"scripts": {"sass": "sass --watch src/scss:build/css", "tarea": "gulp tarea"},

Y luego puedo ejecutar con npm run tarea
